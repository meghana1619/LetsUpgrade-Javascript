There are two types of datatype in js:

  • PRIMITIVE
    • Int

      holds an integer value

    • Float

      holds a string value

    • boolean

      holds either true or false

    • String

      holds a string variable

    • bigInt

      if int size is above 2^53-1 then it is big int

  • REFERENCE
    • Array

      when a value is more than one its index starts from zero

    • Object

      is a key value pair
